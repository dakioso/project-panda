Vad var bra med kursen

  • Upplägget på kursen var riktigt bra och läraren hade ett pedagogiskt lärosätt
  • Läraren förklarade på ett bra sätt så man lätt kunde förstå. Tydliga mål från start vad vi förväntades kunna när kursen var avslutad
  • Föreläsningarna var kanonbra
  • Mitt betyg
  • Allt
  • Att vi har kört projektarbete har gjort att man lärt sig väldigt mycket på vägens gång

Vad har varit dåligt med kursen

  • Jag tycker att det var lite för stressigt med alla uppgifter
  • Det var för mycket föreläsningar
  • För långa föreläsningar så man tappade fokus lätt
  • Läraren var duktig men kunde ha gått in mer invecklat i ämnet, kände att det var alldeles för lätt nu och kursen gav mig inte så jätte mycket
  • För långa föreläsningar, hade vart bättre med flera pauser för att behålla fokus
  • Kursen var för kort, jag känner att jag inte har hunnit lära mig allt som förväntades av mig
